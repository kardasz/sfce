# Symfony Certification Study Guide

A static study guide for the [Symfony certification exam](https://certification.symfony.com/), built with Astro and Tailwind CSS.

Visit the app at [kardasz.github.io/sfce](https://kardasz.github.io/sfce)

## Supported Versions

- [Symfony 8.0](docs/8.0.md) (default)
- [Symfony 7.0](docs/7.0.md)

## Getting Started

```bash
npm install    # install dependencies
npm run dev    # start dev server
npm run build  # production build
```
