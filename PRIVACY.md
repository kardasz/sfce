# Privacy Policy

**Last updated:** April 15, 2026

## Overview

SFCE Learning ([sfcelearning.com](https://sfcelearning.com)) is a static study guide for the Symfony certification exam. We are committed to protecting your privacy.

## Data Collection

This website **does not collect, store, or transmit any personal data** to any server. There are no:

- User accounts or registration
- Cookies (except those set by GitHub Pages infrastructure)
- Analytics or tracking scripts
- Third-party advertising
- Server-side data processing

## Local Storage

The site uses your browser's `localStorage` to save your study progress (checked/unchecked references). This data:

- Is stored **only on your device**
- Is never sent to any server
- Can be cleared at any time through your browser settings
- Contains no personally identifiable information

## Hosting

This site is hosted on **GitHub Pages**. GitHub may collect technical information such as IP addresses in server logs as part of their infrastructure. For details, see [GitHub's Privacy Statement](https://docs.github.com/en/site-policy/privacy-policies/github-general-privacy-statement).

## External Links

This site contains links to external documentation (e.g., symfony.com, php.net). These sites have their own privacy policies, and we are not responsible for their content or practices.

## Changes

If this privacy policy changes, the updated version will be posted on this page with a revised date.

## Contact

If you have questions about this privacy policy, please open an issue on the [GitHub repository](https://github.com/kardasz/sfce).
